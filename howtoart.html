What I Made

I created a website that has a hot air balloon with moving clouds in the background.
Whenever you click on the screen, a color-changing balloon appears where ever you
click. Depending on where the location of the mouse, the balloons will either be
turquoise or color changing. This project, overall, works very well. Everything
that I wanted the website to do that I put in the code works.


In this website, I used many different types of code. My first step was to make the
hot air balloon under the draw function. To make it, I used many different shapes.
I used ellipses to make the balloon part of the hot air balloon. I put multiple
ellipses, each getting skinnier and alternating colors, inside of one perfect circle
ellipse to make the stripes. I used lines to make the ropes. The basket and the mini
baskets on the side of the big basket is made from rectangles.



My second step was to create the moving clouds. To do that, I used a loop, making the
clouds move. I first made a drawCloud function to draw my cloud so it is organized and
I can put it directly into the loop. I used a for loop to make the clouds move to the
right, making the hot air balloon look like it's moving.



My next step after the clouds were to make the balloons that appeared when you click on
the screen. I made a drawBalloon function to make the balloon. I put the drawBalloon
function in a mouseClicked function to make it appear where ever you click on the screen.
To make the balloons change color when the location of the mouse changes, I used an if/else
statement. In it, I made the balloons go from turquiose to flashing different colors. In the
code, I found a problem where the background kept going over each balloon I made, so to fix
it, I created an array to store the data so that the background would not go over the balloons.



My Take Away

What I learned in this project is how to use javascript. I learned how to use many different
codes in javascript, like how to make a loop, function, array, and how to make shapes. I
learned that the loop makes things run for a certain amount of time, but not over time. The
function is to made to perform a particular task. The array is used to store data in your code.
I had to make many decisions on the different functions I made and how I wanted the loop to work.
I had to decide on what I wanted to make interactive. Some things I would do to improve my project
is organize the code a little more, and learn how to use other javascript code to make my art look
better.
