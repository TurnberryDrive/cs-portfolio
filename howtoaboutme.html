    What Did I Create?

    I created a website that shows some of the many different things about me. This website is made up of amny codes that I have put together to create this website.

    What is HTML?

    HTML stands for Hyper Text Markup Langage. It is the standard markuo language for creating web pages. It is the skeleton of your website. The HTML tags are element names that are surrouned by angle brackets.

    Paragraphs, Headers, and Images

    To make my title, I used an <h> tag. To center it, I added a style attribute called text-align: center;. To create the paragraphs, I used the <p> tag. To add an image, I used the src attribute with the <img> tag to put an image from the internet.

    I made the second title by using the <h2> tag. I added a style attribute to change the size and color of the font. To make a list, you use the <ul> tag. The <ul> tag creates bullet points. Whenever you create a list, you always have to use the <li> tag. Without it, you can't make a list.

    What is CSS?

    CSS is the language that describes the style of an HTML document. In the CSS, you use it to add different styles to your text. I took a selector from my HTML and then added declarations in ellipsis. I put a <div> tag around all my <p> tags and gave each of them a different id so that in the CSS, they don't get mixed up.

    What is a navbar?

    A navbar allows you to click on a certain part of the text that takes you to a different page or website. To make a navbar, I had to create a css that would tell what the navbar to do. Once you create the css, you start adding different declarations to it. You would add the declarations to the selectors to change what you want in the website. In the navbar selector I added a background color for the font by adding a declaration(background-color: purple;). I also added text-align: center; to center the navbar.
